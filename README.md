 Task 7: Feature Engineering for Heart Disease Prediction

 Overview

This project implements a feature engineering pipeline for a Heart Disease Prediction dataset. It includes feature preprocessing, baseline model training using a Random Forest Classifier, feature importance analysis, and data leakage detection.

 Features

* Data preprocessing and feature engineering
* Baseline Random Forest model
* Feature importance analysis
* Leakage detection
* Automated execution using `main.py`

 Project Structure

```text
data/
models/
outputs/
feature_engineering.py
train_baseline.py
feature_importance.py
leakage_check.py
main.py
README.md
requirements.txt
```

 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Joblib

 How to Run

`bash
pip install -r requirements.txt
python main.py
```
